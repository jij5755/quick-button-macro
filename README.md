# Quick Button Macro (퀵버튼 매크로)

반복적으로 입력하는 텍스트를 버튼 하나로 자동 입력해 주는 Windows 데스크톱 매크로 프로그램입니다. 사용자가 직접 버튼을 배치하고, 각 버튼에 미리 정해 둔 텍스트를 저장해 두면, 버튼을 클릭할 때마다 지정한 위치에 해당 텍스트가 자동으로 붙여넣어집니다. 정형화된 문구를 자주 입력해야 하는 작업(진료 기록, 상담 응대, 코드 스니펫 등)에 유용합니다.

PyQt5 기반의 GUI 애플리케이션이며, 화면 구성은 한국어로 되어 있습니다.

## 주요 기능

- **클릭 한 번으로 텍스트 자동 입력** — 버튼에 저장한 텍스트를 클립보드에 복사한 뒤, 미리 지정한 마우스 위치를 더블클릭하고 자동으로 붙여넣기(Ctrl+V) 후 Enter까지 실행합니다.
- **자유로운 버튼/텍스트 박스 배치** — 편집 모드에서 위젯을 드래그하여 원하는 위치에 놓고, 크기를 조절할 수 있습니다. 그리드 스냅(10px)을 지원합니다.
- **두 줄 레이블 버튼** — 버튼에 메인 레이블과 보조 레이블을 함께 표시할 수 있습니다.
- **텍스트 박스(주석)** — 글꼴, 글자색, 배경색을 개별 지정할 수 있는 설명용 텍스트 박스를 배치할 수 있습니다.
- **세트(Set) 관리** — 여러 개의 버튼 묶음을 세트로 구성하여 목록에서 전환할 수 있습니다.
- **프리셋(Preset) 저장/불러오기** — 전체 구성을 프리셋 단위로 `presets/` 폴더에 JSON으로 저장하고, 이름 변경·전환이 가능합니다.
- **위젯 복사/붙여넣기 및 다중 선택** — 여러 위젯을 한 번에 선택하여 복사·이동할 수 있습니다.
- **개별 스타일 지정** — 버튼·텍스트 박스별로 크기/글꼴/색상을 따로 설정할 수 있습니다.
- **마우스 목표 위치 설정** — "마우스 위치 설정"을 누르면 3초 후 현재 마우스 포인터 위치가 매크로 입력 대상 위치로 저장됩니다.
- **편집 모드 / 실행 모드 전환** — 편집 모드에서는 매크로가 실행되지 않아 안전하게 레이아웃을 수정할 수 있습니다.

## 동작 방식

버튼을 클릭하면 다음 순서로 동작합니다.

1. 버튼에 저장된 텍스트를 클립보드에 복사
2. 지정된 목표 위치로 마우스 이동 후 더블클릭
3. `Space` → `Ctrl+A`(전체 선택) → `Ctrl+V`(붙여넣기) → `Enter` 순서로 키 입력 자동 실행

> 마우스 및 키보드 자동 제어에는 `pyautogui`를, 클립보드 처리에는 `pyperclip`을 사용합니다.

## 요구 사항

- Windows
- Python 3.x
- 의존 라이브러리:
  - `PyQt5`
  - `pyautogui`
  - `pyperclip`

## 설치 및 실행

```bash
# 의존성 설치
pip install PyQt5 pyautogui pyperclip

# 실행
python main.py
```

> `main.py`는 `main_window.py`와 같은 폴더에서 실행되어야 합니다.

## 빌드 (실행 파일 생성)

[PyInstaller](https://pyinstaller.org/)와 동봉된 `QuickButtonMacro.spec` 파일로 단일 실행 파일(.exe)을 만들 수 있습니다.

```bash
pip install pyinstaller
pyinstaller QuickButtonMacro.spec
```

빌드 결과물은 `dist/QuickButtonMacro.exe`로 생성됩니다.

## 프로젝트 구조

| 파일 | 설명 |
| --- | --- |
| `main.py` | 애플리케이션 진입점. 실행 중 오류 발생 시 설정 파일을 자동 백업합니다. |
| `main_window.py` | 메인 윈도우 및 핵심 로직(매크로 실행, 세트/프리셋 관리, 편집 기능 등). |
| `widgets.py` | 선택 가능한 버튼·텍스트 박스 등 커스텀 위젯 클래스. |
| `dialogs.py` | 버튼/텍스트 박스 설정, 글꼴·색상·크기 지정 등 각종 대화상자. |
| `models.py` | `ButtonSet`, `WidgetClipboard`, `PresetManager` 등 데이터 모델. |
| `constants.py` | 기본 크기, 글꼴, 저장 파일명 등 전역 상수. |
| `QuickButtonMacro.spec` | PyInstaller 빌드 설정. |

## 데이터 저장

- 프리셋은 `presets/` 폴더에 `<프리셋이름>.json` 형식으로 저장됩니다.
- 마지막으로 사용한 프리셋 정보는 `preset_meta.json`에 기록됩니다.
- 구버전 형식(`button_sets.json`)은 실행 시 자동으로 마이그레이션됩니다.
